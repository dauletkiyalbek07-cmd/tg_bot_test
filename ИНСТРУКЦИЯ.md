# 🤖 Инструкция по запуску бота

## ШАГ 1 — Получи токен бота
1. Открой Telegram, найди @BotFather
2. Напиши /newbot
3. Введи имя бота и username
4. Скопируй токен (выглядит так: 7123456789:AAF...)

## ШАГ 2 — Установи Python и библиотеку

### Установка Python (если нет):
Скачай с сайта: https://python.org/downloads

### Установка библиотеки:
Открой терминал (cmd) и напиши:
```
pip install aiogram
```

## ШАГ 3 — Настрой бота (файл bot.py)

Открой файл bot.py и замени:

1. **BOT_TOKEN** — вставь свой токен от @BotFather
   ```python
   BOT_TOKEN = "7123456789:AAF_ТвойТокенСюда"
   ```

2. **CHANNELS** — вставь username своих каналов (без @):
   ```python
   CHANNELS = [
       {"name": "Мой кино канал 🎬", "username": "moy_kanal", "url": "https://t.me/moy_kanal"},
   ]
   ```

3. **ADMIN_ID** — вставь свой Telegram ID (узнать: @userinfobot)
   ```python
   ADMIN_ID = 123456789  # Твой ID
   ```

4. **MOVIES** — добавь свои фильмы:
   ```python
   MOVIES = {
       "001": "🎬 Название фильма (год)",
       "002": "🎬 Другой фильм (год)",
   }
   ```

## ШАГ 4 — Запусти бота

```
python bot.py
```

Если всё верно, увидишь: бот начнёт работать!

## 🌐 Деплой (чтобы бот работал 24/7)

### Бесплатный вариант — Railway:
1. Зайди на https://railway.app
2. Создай новый проект → Deploy from GitHub
3. Загрузи bot.py
4. Добавь переменную окружения: BOT_TOKEN = твой_токен
5. Готово — бот работает постоянно!

### Другие варианты:
- **Render.com** — бесплатно
- **VPS сервер** — платно (~$3-5/месяц), самый надёжный

## ❗ Важно для проверки подписки
Бот должен быть **администратором** в каналах, иначе не сможет проверить подписку.

1. Зайди в настройки канала
2. Администраторы → Добавить администратора
3. Найди своего бота → дай права (достаточно минимальных)
